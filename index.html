For developers and builders

Yale Research Match publishes a normalized lab index as a simple JSON feed. Build new tools, visualizations, and integrations on top of the same data.

Public API endpoints

GET /api/labs_tagged.json full lab index (JSON)
GET /api/labs?q=aging&dept=Chemistry filtered search
GET /api/labs/<id> single lab with notes
GET /api/status health + index size

Schema (labs_tagged.json)

Identity

id, name, pi_name, pi_title, department, school, lab_url, contact_email

Discovery

primary_topics, secondary_topics, tags, methods, skills_preferred, skills_optional

Fit signals

working_style, undergrad_friendly, openings_status, funding_level, openings_notes, min_commitment_weeks, recommended_years

Annotations

student_notes[ ] with id, text, category, status (approved only shown)

Provenance

source_flags: yura_listed, ysm_listed, dept_listed. match_metadata: topic_tokens, last_indexed
